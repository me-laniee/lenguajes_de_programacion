¿Qué es HTML?

HTML (HyperText Mar kup Language) es el lenguaje estándar para crear páginas web. Se utiliza para estructurar el contenido que ves en un n avegador, como textos, imágenes, enlaces y videos.

¿Cómo funciona HTML?

HTML utiliza "etiquetas" que le dicen al navegador cómo mostrar el contenido. Cada etiqueta se escribe entre símbolos de menor y mayor que (< >). Muchas etiquetas vienen en pares: una de apertura y otra de cierre.
